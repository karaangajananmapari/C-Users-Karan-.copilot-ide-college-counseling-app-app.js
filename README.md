# FuturePath College Counseling App

This is a simple starter prototype for a college counseling application for 12th-pass students.

## Included features
- Student quiz for personalized recommendations
- College list with filters
- Shortlist and compare actions
- Local storage for saved colleges
- Ready for real college data replacement

## How to run
Open [index.html](index.html) in a browser, or run a simple local server from this folder.

## Next steps
Replace the sample data in [colleges.json](colleges.json) with your real college list and add your own branding.
